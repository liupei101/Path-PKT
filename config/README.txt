This directory contains the configuration files for model training:
- cfg_sa_base_uni2h_stl.yaml: used for cancer-specific training;
- cfg_sa_base_uni2h_stl_zero_transfer.yaml: used for model transfer test;
- cfg_sa_base_uni2h_stl_moe_pkt.yaml: used for training MoE-PKT models.

Note: these files have been anonymized according to the conference policy, 
      so they could be only used as an example for experiment configuration.
      Complete config files will be made public after acceptance.
